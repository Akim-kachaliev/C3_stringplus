# C3_stringplus

Реализация библиотеки string.h на C (C11, POSIX.1-2017) с поддержкой sprintf и специальных функций.

# Сборка
В корневой директории проекта (где находится src/):

Сборка статической библиотеки
make s21_string.a

Сборка и запуск тестов
make test

Генерация отчёта покрытия gcov (HTML)
make gcov_report

Очистка собранных файлов
make clean

Сборка всего (библиотека + тесты)
make all
Собранная библиотека: src/s21_string.a

# Использование

#include "s21_string.h"

int main() {
    char buffer[100];
    
    // sprintf
    s21_sprintf(buffer, "Hello %s! Number: %d", "World", 42);
    
    // strlen
    size_t len = s21_strlen(buffer);
    
    // memcpy
    char dest[20];
    s21_memcpy(dest, buffer, 10);
    
    // strtok
    char str[] = "one,two,three";
    char *token = s21_strtok(str, ",");
    while (token) {
        printf("%s\n", token);
        token = s21_strtok(NULL, ",");
    }
    
    return 0;
}

Особенности реализации
Префикс s21_ у всех функций

Тип s21_size_t (unsigned long) вместо size_t

Макрос S21_NULL вместо NULL

Платформозависимые сообщения об ошибках (Linux/Mac) для s21_strerror

Google Style для оформления кода

Структурное программирование (без копирования стандартной библиотеки)

# Проверка стиля

./style_test.sh src/
Проверяет все .c и .h файлы через clang-format.

Примечания
Библиотека работает с ASCII (однобайтовые символы)

Ввод через stdin не поддерживается

Запрещено использование sys_nerr, sys_errlist и других системных списков ошибок

Используется своя реализация ошибок (макросы S21_ERRLIST)
